# NLP-Product-Title-Conciseness-Model(BiLSTM-DistilBERT)
E-Commerce product title conciseness classifier framed as a regression task (RMSE evaluation). Multi-input BiLSTM and fine-tuned DistilBERT, both combining text encoding with structured features (price, category, country). Best RMSE: 0.332 with DistilBERT on ~36k Lazada product listings.
